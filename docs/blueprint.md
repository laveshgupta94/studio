# **App Name**: CampusAI

## Core Features:

- User Authentication: Secure login/signup using email/password or Google OAuth with JWT sessions.
- Dashboard: Centralized view of recent activity, progress analytics, AI chat history, upcoming deadlines, and attendance summary.
- AI Chatbot: AI-powered assistant for academic support (explanations, summaries) and college information (exams, events, attendance), processing natural language queries to respond in kind.
- College Data Integration: Admin portal for uploading college data (exam schedules, events, attendance rules) which the AI Chatbot then tool uses to intelligently incorporate into its responses.
- Notifications: Reminders for deadlines, exam notifications, event updates, and attendance alerts via email and browser push.
- Data Analytics: AI analysis of student activity, providing charts, highlighting weak areas, and generating a study consistency score.
- Automation Workflows: Automatic study timetable creation, adjustments based on exam dates, and generation of weekly performance summaries.

## Style Guidelines:

- Primary color: Soft blue (#90EE90) for a calming, academic feel.
- Background color: Light gray (#F5F7FA) for a clean and distraction-free interface.
- Accent color: Vibrant orange (#FF7F50) to highlight important notifications and CTAs.
- Headline font: 'Space Grotesk' sans-serif for a contemporary and slightly techy feel.
- Body font: 'Inter' sans-serif for readability in longer text.
- Use minimalist and modern icons related to academic subjects and college life.
- Clean and spacious layout with large buttons and clear navigation for a student-friendly UI.
- Subtle animations for loading screens and transitions to enhance user experience without being distracting.
